-> Employee Management System

A Spring Boot REST API for managing employee data using CRUD operations.  
This project demonstrates a layered backend architecture with Controller, Service, Repository, and Exception handling.

-> Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA (Hibernate)
- MySQL
- Lombok

-> Features
- Create employee
- Retrieve all employees
- Retrieve employee by ID
- Update employee details
- Delete employee
- Global exception handling

-> API Base URL
http://localhost:8080/api/employees

-> Author
Developed as a backend project to practice Spring Boot and REST API development.
